# Prolog
 Prolog on Common Lisp
 This code is from PAIP (Paradigme of Artificial Intelligence)
 The text for Prolog in PAIP is at Chapter 11 (https://github.com/norvig/paip-lisp/blob/main/docs/chapter11.md)
 and Chapter 12 (https://github.com/norvig/paip-lisp/blob/main/docs/chapter12.md).
 The PAIP program code is found at prolog.lisp, prolog1.lisp, prologc.lisp, prologc1.lisp, prologc2.lisp, and prologcp 
 in https://github.com/norvig/paip-lisp/tree/main/lisp .
 This code by Seiji Koide is, however, slightly changed by using lisp package functionality.
 The change will have a great advantage for the usage of it in Unification Grammar of Chapter 20 and Semantic Web Programing.
